QUEM SOMOS

A BLUM3D é uma empresa criada com fins académicos, para o desenvolvimento de um projeto universitário. Inicialmente trabalhamos para identificar riscos aos quais as pessoas estão vulneráveis ao navegar na internet. A partir dessa identificação procuramos para desenvolver soluções no intuito de suprir essas falhas de segurança na internet.

EQUIPE

Somos uma equipe de estudantes, do curso de graduação em Ciência da Computação, da Pontifícia Universidade Católica de Minas Gerais. Nos unimos no ano de 2021, criando a BLUM3D para o desenvolvimento de um projeto na disciplina “Trabalho Interdisciplinar Aplicações Web”.

O QUE FAZEMOS?

Sabemos que a internet é um lugar não tão seguro para aqueles que nela navegam, então com esse intuito trabalhamos para desenvolver um projeto que poderia suprir essa necessidade. Inicialmente nos focamos em identificar problemas que poderíamos solucionar. A partir disso procuramos desenvolver uma solução para o problema.

Em nosso site é possível encontrar algumas funcionalidades criadas por nos, sendo elas capazes de suprir algumas necessidades. Dentre elas temos o Gerador de Senhas, capaz de criar senhas fortes para o uso do usuário, o Armazenador de Senhas, que pode guardar senhas para que não seja preciso memorizara-las, e também a área de notícias, onde o usuário pode se manter informado.
